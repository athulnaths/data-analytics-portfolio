# Supply Chain Analytics Dashboard | Power BI

## Overview
This project delivers an end-to-end Supply Chain Analytics solution using Power BI. It integrates procurement, production, inventory, logistics, and sales data into a unified analytical model.

---

## Objectives
- Improve supply chain visibility
- Track key performance indicators (KPIs)
- Enable data-driven decision making

---

## Data Model
- Star schema design
- Multiple fact tables:
  - Fact_Sales
  - Fact_Procurement
  - Fact_Inventory
  - Fact_Shipments
  - Fact_Production
- Dimension tables:
  - Dim_Product
  - Dim_Supplier
  - Dim_Facility
  - Dim_Date

---

## Key KPIs
- Revenue & Profit
- OTIF (On-Time In-Full)
- Inventory Turnover
- Fill Rate
- Defect Rate

---

## Tools Used
- Power BI
- DAX
- Excel
